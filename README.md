### 🚀 Featured Projects

- **[Graph Theory Application for Traffic Optimization](https://github.com/kevin-wirya/Braess-Paradox-for-Traffic-Optimization):** Mathematical modeling simulation applying Graph Theory to analyze urban traffic flow, evaluating User Equilibrium vs System Optimal and calculating the Price of Anarchy. (Python)
- **[Speech Deepfakes Detection with FFT](https://github.com/kevin-wirya/Speech-Deepfakes-Detection-with-FFT):** A pure mathematical deepfake detection system utilizing Fast Fourier Transform and complex linear algebra, built without neural networks. (Python, Flask, React, Vite)
- **[Time-Dependent Graph Optimization for Wildfire Suppression](https://github.com/kevin-wirya/spatio-temporal-wildfire-routing):** Interactive web application demonstrating time-dependent pathfinding for wildfire fleets using a modified Haversine heuristic A* search, benchmarking against Dijkstra on 12 historical scenarios. (Next.js, React, Node.js, Leaflet.js)
